Comment récupérer l’entête d’un courrier électronique

Pour analyser un message, GENESIS a besoin de son entête complet.

  1. Ouvrez le courrier électronique avec l'application "Messagerie Thunderbird"
  2. Utilisez le raccourci (Ctrl + U)
  3. Copiez l’intégralité du contenu affiché
  4. Revenez sur GENESIS
  5. Collez le contenu dans le champ « Texte à analyser »

Il faut bien récupérer le courrier initial sur l'appareil de la victime. Si vous transférez le message sur votre boîte professionnelle, les données seront modifiées. Vous pouvez au choix :

  • Ou copier le code source depuis l'appareil de la victime, en affichant l'entête du courrier électronique via le bouton menu "...".
  • Ou télécharger le courrier sur l'appareil de la victime via le bouton "télécharger" de sa messagerie (format .eml), avant de vous l'envoyer.

Comment exploiter un résultat de réquisition

Ce module sert à extraire rapidement des informations depuis un copier-coller de résultat de réquisition, peu importe son format, pour les rendre lisibles et exploitables.

1. Récupérer le contenu

  1. Ouvrez le document (.pdf, .odt, .csv, .txt…).
  2. Sélectionnez la totalité du contenu du résultat (Cltl + A).
  3. Copiez le texte (Ctrl + C).
  4. Collez dans le champ Texte à analyser du module réquisition (Ctrl + V).

2. Choisir les éléments à extraire

Cochez uniquement ce qui vous intéresse :

  • Adresse e-mail : identifiants de comptes, contacts, correspondants.
  • Adresse IP : IP publiques, journaux de connexion, traces, logs.
  • Adresse postale : domiciliation, livraison, facturation.
  • Téléphone : numéros, contacts, identifiants mobiles et opérateur.

3. Lancer l’analyse

  1. Cochez au moins un type d'élément.
  2. Cliquez sur Analyser.
  3. Les résultats apparaissent à droite.
